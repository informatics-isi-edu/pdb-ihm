# Running the archive on the Ubuntu dev instance for the dev catalog 99

Get the `pdb_archive_dev.sh` script from the github repository (`scripts/cron/crontab/pdb_archive_dev.sh`).

As `ubuntu` user, run:

```
sudo pdb_archive_dev.sh
```
